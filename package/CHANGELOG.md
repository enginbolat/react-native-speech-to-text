# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2026-02-15

### Initial Release

- **High-performance Nitro Modules**: Ultra-low overhead speech-to-text bridge.
- **Microphone Recognition**: Real-time STT for iOS and Android.
- **File Transcription**: Support for transcribing audio files on iOS.
- **The Hook Way**: Professional `useSpeechRecognizer` hook with built-in permission handling.
- **Type-Safety**: Comprehensive `api.d.ts` and Nitro-generated spec types.
- **Expo Support**: Built-in config plugin for automated permission handling on iOS and Android.
- **Volume Metering**: Real-time energy levels for voice UI visualizations.
