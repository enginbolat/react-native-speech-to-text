# Example App

This is a demonstration app for `@enginnblt/react-native-nitro-speech-to-text`.

## Running the app

1. **Install dependencies**:

   ```bash
   bun install
   ```

2. **Install pods (iOS)**:

   ```bash
   bun pods
   ```

3. **Start the app**:

   ```bash
   bun start
   ```

4. **Run on a device/simulator**:
   - Press `i` for iOS
   - Press `a` for Android

## Features demonstrated

- Real-time recognition with volume meter.
- Language selection.
- File transcription (select an audio file from your device).
- Error handling.
